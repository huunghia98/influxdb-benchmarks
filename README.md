# Usage
<!-- usage -->
```sh-session
$ npm install -g influxdb-benchmarks
$ influxdb-benchmarks COMMAND
running command...
$ influxdb-benchmarks (-v|--version|version)
influxdb-benchmarks/0.0.0 linux-x64 node-v11.3.0
$ influxdb-benchmarks --help [COMMAND]
USAGE
  $ influxdb-benchmarks COMMAND
...
```
